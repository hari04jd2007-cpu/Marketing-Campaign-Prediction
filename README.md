# Marketing Campaign Response Prediction

A Machine Learning and Streamlit web application that predicts whether a customer is likely to respond to a marketing campaign.

## Features

* Dashboard Analytics
* Admin Page
* Decision Tree Model
* Random Forest Model
* SVM Model
* XGBoost Model
* Accuracy Comparison
* Customer Response Prediction
* Streamlit User Interface

## Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-Learn
* XGBoost

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
